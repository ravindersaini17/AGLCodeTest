# AGL CodeTest
AGL programming challenge.

# Requirement
You need to write some code to consume the json and output a list of all the cats in alphabetical order under a heading of the gender of their owner.

    Example:

        Male
            Angel
            Molly
            Tigger

        Female
            Gizmo
            Jasper

# Notes

    Submissions will only be accepted via github or bitbucket.
    Use industry best practices.
    Use the code to showcase your skill.

# Run Code In Visual Studio

    Download the code and open solution AGL.CodeExercise.App.sln file in Visual studio.
    Set AGL.CodeExercise.App as startup project.
    Build the code and make sure Nuget Package Restore has run.
    Run and check console output.

# Required Tools:
    Visual Studio Code
    .Net Core 3.1
